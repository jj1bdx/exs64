# exs64: xorshift64star PRNG for Erlang

xorshift64star is a 64-bit PRNG of (2^64-1) period.

See <http://xorshift.di.unimi.it/> for the further details.

## Author

Algorithm by Sebastiano Vigna, made public domain.
